# Next Word Prediction Using LSTM

## 📌 Project Overview

This project implements a **Next Word Prediction** model using **Deep Learning (LSTM)** and **TensorFlow/Keras**. The model is trained on movie titles from the TMDB 5000 Movies dataset and predicts the most probable next word based on the input sequence.

The project demonstrates Natural Language Processing (NLP), text preprocessing, tokenization, sequence generation, and language modeling using Long Short-Term Memory (LSTM) networks.

---

## 🚀 Features

- Text preprocessing
- Tokenization using TensorFlow/Keras
- Sequence generation
- LSTM-based Deep Learning model
- Next word prediction
- Model training and evaluation
